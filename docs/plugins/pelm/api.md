# PELM API Endpoints

| Endpoint | Description |
|---------|-------------|
| `/pelm/status` | Current risk + metadata |
| `/pelm/run` | Execute PELM scan |
| `/pelm/trend` | Risk trend data |
| `/pelm/snapshots/list` | List snapshots |
| `/pelm/snapshots/get/{name}` | Retrieve snapshot |
| `/pelm/snapshots/diff` | Compare snapshots |
| `/pelm/regression` | Regression analytics |
| `/pelm/governance/repair` | Repair governance state |
| `/pelm/report/html` | HTML report |
| `/pelm/report/markdown` | Markdown report |
