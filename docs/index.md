## What This Project Provides

### **Operator‑Grade Build System**
A deterministic Makefile with:

- Bootstrap workflows  
- CI gates  
- Docker workflows  
- Formatting, linting, testing  
- Full uninstall suite  

### **Structural Validators**
Ensuring the project never drifts:

- Makefile integrity validator  
- Project structure validator  
- Python import validator  

### **Cross‑Platform Operator Tools**
- Linux/macOS scripts under `scripts/`  
- Windows PowerShell mirrors under `scripts/windows/`  
- Unified `ops` wrapper  
- TUI operator menu  

### **Documentation Suite**
Everything you need to understand, operate, and extend the console.

## Project Goals

The SSRF Command Console is built to:

- Provide deterministic, observable workflows  
- Support modular backend development  
- Enforce structural integrity through validators  
- Offer a complete operator experience across platforms  
- Serve as a foundation for advanced SSRF research and tooling
