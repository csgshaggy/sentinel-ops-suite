# Development Manual

This guide covers developer workflows.

## Setup

Choose one environment:

### pip

make bootstrap

## uv

make uv-bootstrap

## Poetry

make poetry-bootstrap


## Code Quality

- Format: `make format`
- Lint: `make lint`
- Test: `make test`

## Rebuild Workflow

make rebuild


## Plugin Development

Plugins live under:

tools/plugins

List them:

make plugins
